# 🚀 Amit Portfolio

A futuristic cyberpunk-style personal portfolio website built using HTML, CSS, and JavaScript with modern UI effects, animations, glassmorphism, neon glow styling, and responsive design.

**✨ Features**

 * 🌍 Floating Earth Animation
 * ✨ Particle Background Effect
 * ⌨️ Typing Animation
 * 🟢 Neon Glow UI
 * 🪟 Glassmorphism Design
 * 🎨 Modern Cyberpunk Theme
 * 📱 Fully Responsive Layout
 * 🚀 Scroll Reveal Animations
 * 💡 Mouse Glow Effect
 * 🔗 Social Media Links
 * 📂 Project Showcase Section
 * 📄 Resume Download Button
 * 📬 Working Contact Form

---

 **🛠️ Built With**

* HTML5
* CSS3
* JavaScript
* AOS Animation Library
* Particles.js
* Typed.js
* Font Awesome

**⚡ Installation**

Clone the repository:

git clone https://github.com/prodevamit/portfolio.git

Open the project folder:

cd portfolio

Run using Live Server in VS Code.
